# C++ Moderno

Bienvenido al repositorio oficial del libro **"El lenguaje de programación c++"**.  
Este proyecto reúne ejemplos, ejercicios y recursos complementarios para aprender y dominar las características más recientes del lenguaje C++.

---

## Objetivo
El propósito de este libro es guiar a programadores —desde principiantes hasta avanzados— en el uso de las herramientas modernas de C++ (C++11, C++14, C++17, C++20, C++23 y más).  
Aquí encontrarás explicaciones claras, buenas prácticas y ejemplos prácticos que reflejan cómo se programa en C++ hoy en día.

---

## Contenido del repositorio
- **Ejemplos de código**: fragmentos que ilustran conceptos clave.
- **Ejercicios prácticos**: retos para reforzar el aprendizaje.
- **Notas adicionales**: material complementario al libro.
- **Recursos externos**: enlaces a documentación oficial y comunidades.

---

## Temas principales
- Sintaxis moderna y mejoras del lenguaje.
- Uso de `auto`, `constexpr`, `expresiones lambda`.
- Programación genérica con *plantillas* y *conceptos*.
- Manejo de memoria inteligente con *smart pointers*.
- Programación concurrente y paralela.
- Buenas prácticas y patrones de diseño en C++ moderno.
